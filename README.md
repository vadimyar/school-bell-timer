# 🕰️ Звонилка для уроков (School Bell Timer)

Простое приложение на Python + PyQt5 + matplotlib + pygame, которое отображает текущий урок или перемену в виде круговой диаграммы и проигрывает разные звуки при их начале.

> 💡 Идеально для школьных компьютеров, учеников и учителей!  
> ✅ Работает только на Linux (требуется свободное ПО и окружение X11/Wayland с поддержкой PyQt).

## ✨ Возможности

- Чтение расписания из текстового файла (`schedule.txt`)
- Автоопределение текущего урока/перемены по времени
- Круговая диаграмма с обратным отсчётом
- 🔔 Звук `bell.wav` — при начале **урока**
- 🔊 Звук `bell2.wav` — при начале **перемены**
- Окно **без рамки**, **поверх всех окон**
- Прозрачный фон (только диаграмма и текст видны)
- Перетаскивание окна **любой частью интерфейса**
